interactivity

A collection of interactivity-oriented samples for student use. Most of these are drawn from existing examples, but simplified.

basic

  1. mobile: puts your mobile web browser in fullscreen mode
  2. watch: a round clipping mask that emulates a smart watch display

pointer events

  1. playground: lists all pointer interactions found
  2. pressure: measures the pressure upon touch input (fakes it on non-compatible devices)
  3. track: elements will track pointer movements

computer vision

  1. camera: rendering a video stream from a camera and capturing frames
  2. camera-pixels: processing pixels from a camera stream
  3. camera-clmtrackr: face tracking. basic - emotion classification
  4. camera-deeplearn: object detection

physical

  1. ws-serial-bridge: demonstrates bi-directional serial communication with an Arduino and web app
